Why pvm?

⚡

One command install

Type pvm install 8.3 and you're done. Auto-resolves the latest patch version.

🔄

Instant switching

Switch PHP versions in your current shell. No terminal restart needed.

📁

Auto-switch per project

Add a .php-version file. pvm switches automatically when you enter the directory.

🖥️

Cross-platform

macOS, Linux (build from source), and Windows (pre-built binaries). Works everywhere.

🩺

Built-in doctor

Run pvm doctor to check all dependencies before installing. No guessing.

🐚

Shell integration

First-class support for Bash, Zsh, Fish, and PowerShell. Auto-loads on terminal start.
